# raycast-scripts

Personal [Raycast](https://raycast.com) script commands.

## Scripts

| Script | Description |
|---|---|
| **Toggle Grayscale** | Toggle grayscale mode on all displays via [BetterDisplay](https://github.com/waydabber/BetterDisplay) |

## Setup

1. Install [BetterDisplay](https://github.com/waydabber/BetterDisplay) and `betterdisplaycli` (`brew install waydabber/betterdisplay/betterdisplaycli`)
2. Clone this repo
3. In Raycast, go to Extensions > Script Commands > Add Script Directory and point it at the cloned folder
